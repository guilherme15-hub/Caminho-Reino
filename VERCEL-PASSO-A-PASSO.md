# 📱 Guia Passo a Passo - Deploy no Vercel

## Você já tem conta no Vercel ✅

Agora siga exatamente esses passos:

---

## **PASSO 1: Acessar Dashboard do Vercel**
1. Vá para https://vercel.com/dashboard
2. Você deve ver um botão **"Add New..."** ou **"New Project"** no canto superior

---

## **PASSO 2: Importar do GitHub**
1. Clique em **"Add New..."** → **"Project"**
2. Você verá a opção: **"Import an existing project"**
3. Clique em **"Continue with GitHub"**

---

## **PASSO 3: Selecionar o Repositório**
1. Uma popup perguntará: **"Select a Git Repository"**
2. Procure por: **`Caminho-Reino`** (ou **`guilherme15-hub/Caminho-Reino`**)
3. Clique no repositório para selecioná-lo

---

## **PASSO 4: Configurar o Projeto**
A Vercel deve detectar automaticamente:
- **Framework Preset**: Vite ✓
- **Root Directory**: ./ ✓
- **Build Command**: `npm run build` ✓
- **Output Directory**: `dist` ✓
- **Install Command**: `npm install` ✓

Se estiver correto, não precisa mexer em nada!

---

## **PASSO 5: Deploy**
1. Clique no botão **"Deploy"** (botão azul grande)
2. Aguarde a compilação (vai mostrar logs verde)
3. Quando terminar, você verá: **"Congratulations! Your project has been successfully deployed"**

---

## **PASSO 6: Acessar o Jogo**
1. A Vercel vai gerar uma URL como: 
   ```
   https://caminho-reino.vercel.app
   ```
   (O nome pode ser diferente)

2. Clique no link e pronto! 🎉

---
